## 주제 및 기획의도

- 주제: 레시피버전 인스타그램
- 기획의도 : 코로나-19 시기 가운데 가족들과 함께 집에서 식사하는 시간이 늘어나면서 자신의 특별한 레시피를 공유하고 자신의 삶을 공유 하는 네트워크 구축

## 기능 개요

- 회원가입, 로그인 기능
- 게시판 기능
- 댓글 기능
- 사용자 블락 및 팔로우 기능
- 사용자 간의 메세지 보내기 기능

## Non-Functional Issues

```
Web Service: Firebase
Front-end Framework: React Native
```
